# tno-assets

Static image assets used by the [TNO Superevent Maker](https://github.com/JacqueChirac) — a fan-made tool for generating TNO-styled superevent images.

## Attribution

These assets are **not original work**. They are derived from:

- **Hearts of Iron IV** — © Paradox Interactive AB
- **The New Order: Last Days of Europe** — a fan-made modification for HOI4, owned by its community development team. Workshop ID [2980739000](https://steamcommunity.com/sharedfiles/filedetails/?id=2980739000)

This repository exists solely as an asset-hosting layer for the Superevent Maker app (served via jsDelivr). It is not affiliated with Paradox Interactive or the TNO mod team.

## Contents

- `leaders/` — Leader portraits
- `flags/` — Country flags
- `spirits/` — National spirit icons
- `focus/` — Focus tree icons
- `economy/`, `ministers/`, `ideologies/`, `achievements/`, `event_pictures/`, `superevent_pictures/`, `texticons/`, `ui/`, `overlays/`, `news/`, `fonts/`, `background/`, `helpers/`, `leaders_bg/` — category-specific assets
